PID 控制：怎么让车沿着黑线走？

两层 PID 套在一起：外层决定"往哪拐"，内层决定"跑多快"
串级 PID 结构
方向环（外环 · 位置式）
输入 line_pos（灰度偏移，-100～+100）→ 输出 turn_amount（转向量）
偏右就往左打，偏左就往右打。就这一句话
▼ turn_amount 叠加到两轮速度上 ▼
速度环 ×2（内环 · 增量式）
左轮目标 = base + turn  |  右轮目标 = base − turn
PID 算出实际 PWM → 四路编码器驱动 → 520 电机 → 编码器反馈 → 闭环
增量式 PID 公式
Δu = Kp(ek − ek−1) + Ki·ek + Kd(ek − 2ek−1 + ek−2)
uk = uk−1 + Δu  （当前输出 = 上次输出 + 增量）
三个参数一句话
Kp
大了：车左右扭
小了：纠正太慢跑偏
Ki
大了：速度过冲
小了：永远差一点
Kd
大了：电机嗡嗡抖
小了：急弯反应不过来
为什么用增量式而不是位置式？
  • 天然抗积分饱和——输出每次只变一点点，不会累积到飞出去
  • 直道切弧线时平滑过渡——速度突变时增量式不会"窜"
  • M0+ 随便算——只存最近 3 次误差，一个周期就算完
