# 2024 年电赛 H 题：自动行驶小车

一份给电赛小白的实战项目文档。基于 MSPM0G3507 天猛星，从看懂题目到车跑完 4 圈。

## 打开即用

https://xin360047-cmyk.github.io/racecar-2024h/

## 适合谁

刚学完 STM32 基础、HAL 库、C++ 面向对象，准备打电赛但不知道从哪下手。

## 包含什么

按时间线组织，跟着做就行：

| 步骤 | 内容 |
|------|------|
| 0 | 硬件模块清单 + 装 CCS + SysConfig 配置 |
| 1 | 赛题原文 + 场地示意 + 四问要求 + 评分表 |
| 2 | 分层架构 + 文件结构 |
| 3 | 路径状态机 + 串级 PID |
| 4 | 代码骨架（每段代码有解释） |
| 5 | PID 调参方法 + 故障速查表 |
| 6 | 4 天实战检查清单 |

## 硬件

- MSPM0G3507 天猛星最小系统板
- 8 路灰度传感器
- 四路编码器电机驱动模块
- L 型 520 减速电机 ×2
- MPU6050 陀螺仪
- 0.96" OLED（I2C）

## 本地部署

```bash
git clone https://github.com/xin360047-cmyk/racecar-2024h.git
cd racecar-2024h
# 直接用浏览器打开 index.html
```
